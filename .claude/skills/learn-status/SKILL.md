# /learn-status — 学习进度总览

只读命令：扫描 `topics/*/state.md`，汇总输出，不修改任何文件、不进入学习流程。

## 输出格式

按「进行中」「已结题」分组，各输出一张表：

| 主题 | 目标层级 | 当前阶段 | 模块进度 | 最后学习 |
|------|----------|----------|----------|----------|
| RAG 检索增强生成 | L2 能讲 | ③ 循环学习 | 2/4 | 2026-06-15 |

- 模块进度 = state.md 中已勾选模块数 / 总模块数（②建图前显示 `-`）
- `topics/` 为空时：说明还没有学习主题，提示用 `/learn <主题>` 开始
- 结尾提示：可用 `/learn` 继续进行中的主题
